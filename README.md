## Selected Projects

### 🔁 Manufacturing Process Simulation & Optimization
Multi-stage job flow simulation with constraint-based optimization for manufacturing throughput. Built discrete-event simulation models integrated with optimization solvers to identify bottlenecks, reduce cycle time, and maximize resource utilization across production lines.

**Stack:** Python · SimPy · PuLP / OR-Tools · Pandas · Matplotlib

---

### ⚡ Electricity Futures Trading Insight Tool
Multi-signal trading analytics system for electricity futures contract purchasing. Ingests and aligns forward power curves, natural gas prices, weather forecasts, and grid congestion indicators — heterogeneous sources at different frequencies — onto a common time index. Walk-forward validation framework with strict look-ahead bias controls. Decision-support interface with confidence intervals designed for trader interpretability.

**Stack:** Python · Pandas · Scikit-learn · XGBoost · Plotly

---

### 🔋 Energy Demand & UFG Anomaly Detection Pipeline
End-to-end anomaly detection system for Unaccounted-For Gas (UFG) — a residual derived from metered send-out minus customer consumption. Built to distinguish genuine demand anomalies from data pipeline failures and billing cycle artifacts. Rolling z-score model tuned for noisy residuals across heterogeneous customer segments. Medallion architecture (Bronze → Silver → Gold) with Python pre-scoring surfaced through Power BI for near real-time operational flagging.

**Stack:** Python · PySpark · Azure Databricks · Microsoft Fabric · Power BI · Delta Lake

---

### 📈 Multi-Domain Forecasting Systems
Time-series forecasting across energy demand, retail sales, and commercial load — handling non-stationarity, external regressors (weather, price signals, calendar effects), and sparse data regimes. Built retraining pipelines with automated model staleness detection.

**Stack:** Python · Prophet · XGBoost · Scikit-learn · Azure ML · MLflow
